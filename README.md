# SafeMed Agenda

Sistema de agendamento e gestão de atendimentos para clínicas de pequeno porte.

## Funcionalidades

- Cadastro de pacientes
- Cadastro de profissionais
- Consulta da agenda
- Agendamento de consultas
- Remarcação de consultas
- Cancelamento de consultas
